10/07/2017, HiKV: A Hybrid Index Key-Value Store for DRAM_NVM Memory Systems, ATC'17.
Summary: This paper talks about Indexing efficiency. Hash behaves the best among Hash, SkipList, B+-Tree for individual operations: PUT/GET/DELETE/UPDATE.
However, B+-Tree behaves the best for the bulk operations such as SCAN.
In order to make use of advantages of both Hash and B+-Tree, the authors design a hybrid indexing structure for DRAM_NVM memory systems.
The B+-Tree is put in memory because of its frequent updates to serve SCAN operation, and Hashing is put in NVM to serve the individual operations.
